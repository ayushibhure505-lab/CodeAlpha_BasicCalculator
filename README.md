# CodeAlpha_BasicCalculator
## Description
This project is developed as part of the CodeAlpha C Programming Internship. It performs basic arithmetic operations using switch case statements.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by Zero Handling

## Technologies Used
- C Programming Language

## Concepts Used
- Switch Case
- Conditional Statements
- User Input Handling

## How to Run
1. Compile the program:
   gcc calculator.c -o calculator

2. Run the program:
   ./calculator
